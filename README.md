## 1. Virtual Machine Deployment (VirtualBox)

![VirtualBox Setup](images/virtualbox-setup.png)

In this step, I created and configured a virtual machine using Oracle VirtualBox to simulate a real enterprise environment. I installed Windows Server 2022 to serve as the foundation for my Active Directory lab.

This setup allows me to safely practice IT help desk and system administration tasks in a controlled environment.
